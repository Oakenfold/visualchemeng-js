Usage

Click a boundary to activate it. Drag and drop boundaries to control their position. Use the buttons to add and remove the boundaries.

Keyboard functionality: Right arrow = increase boundary width. Left arrow = decrease boundary width. Up arrow = increase boundary height. Down arrow = decrease boundary width. R = Rotate boundary clockwise. L = Rotate boundary anti-clockwise.
